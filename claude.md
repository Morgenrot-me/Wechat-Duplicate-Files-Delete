# 本地项目规范

## 项目定位

本项目是一个基于 PyQt5 的单机桌面工具，用于识别并删除微信类重复文件。当前项目采用单文件主程序结构，`main.py` 为运行时唯一真源。

## 修改原则

1. 所有安全规则优先于功能便捷。
2. 只有满足“尾部编号候选 + 原文件存在 + 内容哈希一致”的文件才允许进入回收站。
3. 只删除普通文件，不删除目录。
4. GUI 行为和删除规则以 `main.py` 为准，`ui.ui` 当前仅作为历史设计稿参考，不作为运行时加载来源。
5. 修改代码后必须同步更新对应文档。

## 文件约定

- `main.py`：主程序入口、界面定义、删除逻辑。
- `build.bat`：本地打包脚本。
- `README.md`：用户使用说明。
- `根目录说明.md`：根目录文件说明。
- `verify.py`：轻量验证脚本。

## 验证要求

1. 取消目录选择时不得报错，不得删除任何文件。
2. 递归与非递归两种模式都要可验证。
3. 必须验证同名尾号文件在“同内容”和“不同内容”两种情况下的行为。
4. 打包后 GUI 必须能给出可见结果，不依赖控制台输出。
